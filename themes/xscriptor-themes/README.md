<h3>Enable Icon Theme</h3>
<ol>
  <li>Open Command Palette (<code>Ctrl+Shift+P</code> or <code>Cmd+Shift+P</code>).</li>
  <li>Run <code>Preferences: File Icon Theme</code>.</li>
  <li>Select one Xscriptor icon theme.</li>
</ol>

<pre><code>{
  "workbench.iconTheme": "x-icons"
}
</code></pre>

<h3>Enable Product Icon Theme</h3>
<ol>
  <li>Open Command Palette (<code>Ctrl+Shift+P</code> or <code>Cmd+Shift+P</code>).</li>
  <li>Run <code>Preferences: Product Icon Theme</code>.</li>
  <li>Select one variant:</li>
</ol>

<ul>
  <li><code>x Product Icons</code> (glyph variant)</li>
</ul>

<pre><code>{
  "workbench.productIconTheme": "x-product-icons"
}
</code></pre>

<h2>Installation</h2>
<ol>
  <li>Open Extensions view (<code>Ctrl+Shift+X</code>).</li>
  <li>Search for <code>X Themes</code> or <code>xscriptor</code>.</li>
  <li>Install and activate your preferred theme.</li>
</ol>

<p>Terminal install (VSIX):</p>
<pre><code>code --install-extension xscriptor-themes-1.1.0.vsix</code></pre>
