<h1 align="center">UI Mods</h1>

<p>
  Lightweight interface tweaks for VS Code using CSS and JS injection.
</p>

<p>
  Requirement: install the <strong>Custom UI Style</strong> extension before applying these mods.
</p>

<h2>Available Packs</h2>
<ul>
  <li><a href="./xscriptor-ui/">xscriptor-ui/</a>: Xscriptor-focused UI preset files.</li>
  <li><a href="./x-ui/">x-ui/</a>: alternative UI preset files.</li>
</ul>

<h2>Included Files</h2>
<ul>
  <li><code>settings.json</code>: sample profile configuration.</li>
  <li><code>rounded.css</code>: rounded borders and softened corners.</li>
  <li><code>quickcenter.css</code> and <code>quickcenter.js</code>: quick command area alignment and behavior tweaks.</li>
</ul>

<h2>Usage Notes</h2>
<ul>
  <li>Copy the files you need from either <code>xscriptor-ui/</code> or <code>x-ui/</code>.</li>
  <li>Apply them through your Custom UI Style workflow.</li>
  <li>Reload VS Code after changes for consistent rendering.</li>
</ul>
